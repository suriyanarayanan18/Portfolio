Streaming Platform User Analysis

A comparative analytics project analyzing user behavior across Netflix and Amazon Prime to understand how demographics, subscriptions, and device usage shape engagement patterns.

Python Pandas NumPy Matplotlib Seaborn Exploratory Analysis

Analysis focus

This project investigates how users interact with streaming platforms and where behavioral differences emerge between Netflix and Amazon Prime.

  • Demographic patterns across age and gender segments
  • Subscription preferences (monthly vs annual)
  • Device usage trends and viewing habits
  • Behavioral contrasts between platforms
  • Cleaned outputs designed for reuse and reporting

Project design

The workflow was intentionally structured to move beyond notebooks into a reproducible, script-based analysis.

Data source
Kaggle (public datasets)
Execution
src/run.py
Structure
Notebooks → Reproducible code
Outputs
Cleaned CSVs & summaries

Why it matters

This project reflects real-world analytics practices by separating exploration from execution and producing results that can be rerun, shared, and extended.
